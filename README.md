# Gym Management System

This is a basic Gym Management System website developed using Flask, HTML, and CSS.

## Features
- Home page with welcome message and gym highlights
- About Us page describing the gym's mission, team, and facilities
- Membership page showing available membership plans

## Technologies Used
- Python (Flask)
- HTML
- CSS
